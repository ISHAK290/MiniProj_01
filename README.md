# MINI PROJECT #01: Linux Sysadmin Automation Kit [Best done on: Day 1]

**Project Brief:** Write a bash script that performs automated server health checks — checks disk space, RAM usage, running services, and sends a formatted report to a log file. Add a cron job to run it every hour.

**Skills Applied:** Linux, Bash, Cron, Logging

**Success Outcome:** A deployable monitoring script that a sysadmin would actually use.
